### 3. 비정상 감지 배지
- ✅ 비정상 이벤트 쌍에 경고 배지 표시
- ✅ Scenario ID 및 설명 표시
- ✅ InputType 불일치 배지 표시

### 4. UI 개선
- ✅ 각 쌍에 헤더 추가 (Pair #N, 타임스탬프 정보)
- ✅ 에디터 이벤트 섹션 스타일링
- ✅ 스크롤 가능한 히스토리 컨테이너
- ✅ 빈 상태 처리

## 잠재적 개선 사항

1. **타입 안전성 강화**
   - `plugin.getEvents()`의 반환 타입을 명시적으로 `PluginEvent[]`로 처리
   - 현재는 `any[]`로 처리되지만 실제로는 `PluginEvent[]`를 반환

2. **성능 최적화**
   - 많은 이벤트 쌍이 있을 때 가상 스크롤링 고려
   - 현재는 DOM에 모든 쌍을 렌더링하므로 성능 이슈 가능성

3. **사용자 경험**
   - 각 쌍을 접기/펼치기 기능 추가 고려
   - 에디터 이벤트 섹션 기본적으로 접혀있도록 옵션 추가

4. **디버깅**
   - 각 쌍에 대한 상세 정보 보기 기능 추가 고려
   - 이벤트 쌍 클릭 시 상세 정보 모달 표시
